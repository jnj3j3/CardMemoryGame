# Memory Card Game
## 소개
> Memory Card Game은 엎어진 카드들 중 같은 카드를 맞추는 게임입니다.
>>한번에 2개를 고를 수 있으며 고른 2개의 카드가 같지 않으면 다시 엎고 같으면 그대로 놓아 마지막으로는 모든 카드가 엎어져있지 않으면 이기는 게임입니다.
